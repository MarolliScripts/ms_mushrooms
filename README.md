# Simple hallucinogenic mushrooms collecting 🍄
Our new script allows you to easily collect hallucinogenic mushrooms.

## Preview:

## What do I need?
- ESX Framework
- ox_lib
- ox_inventory
- ox_target

# Instalation
1. Go trough ox_inventory and add there all from "INSTALL" folder!
2. Go trough server.cfg and add there "ensure ms_mushrooms"
3. Enjoy!
